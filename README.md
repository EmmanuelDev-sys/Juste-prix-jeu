# Juste Prix Jeu 🎯

Un petit jeu amusant où tu dois deviner un nombre entre **1 et 100** !

**Technologies utilisées** : HTML + CSS + JavaScript

### Comment jouer ?
- Entre un nombre
- L’ordinateur te dit si c’est trop bas ou trop haut
- Essaie de trouver le bon nombre en le moins de tentatives possible !
